# 项目路线与优先指令目录

本目录存放了关于项目发展路线、阶段需求背景以及用户针对特定开发阶段给出的权威优先决策指示。

---

## 路线图文档索引

1. **`USER_DIRECTIVES.md` (用户优先指令)**:
   - *说明*: 记录项目所有者对当前开发周期的最高优先判断与技术偏好（如本次 EMBEDDER MONITOR 侧边栏的设计规范、仿真机制与协议架构）。
   - *用法*: 存在方案决策不确定或旧计划与最新意图冲突时，**必须以此文件为最高决策依据**。

2. **`ROADMAP.md` (项目演进路线图)**:
   - *说明*: 描绘了项目自 v0.2.0 起的四个核心演进阶段（基础 ACP 客户端 $\rightarrow$ EMBEDDER MONITOR 串口与模拟器 $\rightarrow$ 多卡片多协议融合 $\rightarrow$ 生产级固件工具链与标准化规格）。
   - *用法*: 用于对标宏观演进趋势和未来特性规划。
